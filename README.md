# SlovakiaElectionsTool
A tool to visualise Slovakian 2023+2020 election results (+ more later possibly) down to the municipality. Includes an election night simulation.

## Features
- A highly interactive display of Slovak legislative elections in the years 2023 and 2020.
- Detailed map down to the municipality.
- Advanced election night simulation with adjustable parameters
- Poll menu
- Visually appealing UI
